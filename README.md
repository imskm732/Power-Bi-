# AtliQo 5G Impact Analysis Dashboard

AtliQo, a telecom provider in India, launched 5G in May 2022. This dashboard helps management assess the post-5G impact on users, revenue, and market share, guiding decisions to regain growth.

---

- Matched period analysis shows consistent user drops month-to-month after 5G, confirming sustained churn.
